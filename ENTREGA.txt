🚀 COMO COMEÇAR (5 MINUTOS)

1. Clone o projeto
   cd /opt
   git clone <seu-repo> vnstat-dashboard
   cd vnstat-dashboard

2. Execute setup interativo
   bash setup.sh

3. Acesse a dashboard
   http://seu_servidor
   Login: admin / (senha que você digitou)

4. Instale coletores nos hosts
   scp -r collector/ pi:/tmp/
   ssh pi 'cd /tmp/collector && sudo bash install-collector.sh pi http://100.x.x.4 CHAVE eth0'
   (Repetir para note e ser4ph-arm)

5. Aguarde ~10 minutos
   Dados começam a aparecer automaticamente

🎯 DECISÕES TÉCNICAS

Flask over FastAPI         → Familiaridade com seu stack
MariaDB over SQLite        → Escalabilidade
JWT em cookies             → CSRF-safe, SameSite
Collector push model       → Agent leve, não precisa SSH
Chart.js                   → Sem dependencies frontend
Docker Compose             → Zero-config, reproducible
Nginx reverse proxy        → TLS + gzip + caching
